### 系统简介

> 本系统可做为毕设项目，可提供完整的 PPT，以及论文等材料。

系统基于 Spring Boot 和 Ant Design Pro ，主要功能有图表、文章、回答生成，以及个人信息管理，管理员管理，可以对用户生成的图表、文章、回答进行管理。
